# 💧 Automatización de Datos Hídricos - IDEAM

Este proyecto de pregrado en **Ingeniería Civil** (Universidad de la Costa) automatiza la extracción y estructuración de datos de precipitación desde la API de Datos Abiertos Colombia.

## 🚀 Versiones Incluidas
* **Dinámica (`extractor_dinamico.py`)**: Consulta en tiempo real los departamentos y años disponibles en la base de datos del IDEAM.
* **Estática (`extractor_estatico.py`)**: Versión optimizada con listas fijas para un arranque inmediato.

## 📦 Instalación y Uso
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar cualquiera de las dos versiones: `python nombre_del_archivo.py`
